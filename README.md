# Portfolio of Projects

### [Pathfinder](pathfinder/README.md): Mini OpenStreetMap Directions UI

> [!NOTE]
> A tiny, single-file OpenStreetMap directions interface using Leaflet,
> Tailwind, and the OSRM demo routing server. Includes a rounded modern UI,
click-to-set coordinates, zoom/pan, and auto-fitting route display. Demo version
> hosted [here](https://allovelle.github.io/fathom/pathfinder/index.html).
